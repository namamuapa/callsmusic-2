# Calls Music 2 — The first open-source PyTgCalls based project

##### Note that “Beta” doesn’t mean it’s not stable or it’s under construction, it’s just an identifier.

## Requirements

- FFmpeg
- Node.JS 15+
- Python 3.7+

## Deployment

### Config

Copy `example.env` to `.env` and fill it with your credentials.

### The good way

1. Install Python requirements:
   ```bash
   pip install -U -r requirements.txt
   ```
2. Run:
   ```bash
   python main.py
   ```

### Using Docker

1. Build:
   ```bash
   docker build -t musicplayer .
   ```
2. Run:
   ```bash
   docker run --env-file .env musicplayer
   ```

[Click here](https://heroku.com/deploy?template=https://github.com/namamuapa/callsmusic-2_heroku/)

## Commands

| Command | Description                                  |
| ------- | -------------------------------------------- |
| /play   | play the replied audio file or YouTube video |
| /pause  | pause the audio stream                       |
| /resume | resume the audio stream                      |
| /skip   | skip the current audio stream                |
| /stop   | clear the queue and stop the audio stream    |
